Posting checklist

把信息写得像“可执行任务”，就会更容易匹配到人。

  1. 一句话标题 + 明确结果
  2. 写清 输入 / 输出 / 验收标准
  3. 写清可用的时间窗口与响应方式
  4. 写清你能提供什么（署名 / 经费 / 资源）
